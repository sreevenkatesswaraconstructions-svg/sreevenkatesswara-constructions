# Sree Venkatesswara Constructions & Interiors — Starter

This workspace contains a starter Next.js + Tailwind CSS scaffold for a premium multi-page website.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm run start
```

Deployment suggestions:

- Deploy to Vercel: connect the repo, set `NEXT_PUBLIC_BASE_URL` environment variable to your domain.
- For CMS: integrate Sanity, Contentful, or Strapi and replace the static `data/` files with CMS queries.

Files created include pages for Home, About, Services, Projects, Why Us, Blog and Contact, plus core components and styles. Use the `data/` folder for initial content and replace with CMS data when ready.
